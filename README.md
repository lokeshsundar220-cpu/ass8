# AI Mock Interviewer - Web Developer

This project is a role-specific AI mock interviewer built using CrewAI and Streamlit.

## Features
- Asks Web Developer interview questions
- Provides feedback after each answer
- Interactive Streamlit UI

## Tech Stack
- Python
- CrewAI
- Streamlit

## Run the project

```bash
pip install -r requirements.txt
streamlit run app.py
